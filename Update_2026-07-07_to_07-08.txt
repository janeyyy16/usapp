DARKGLASS HUB SUITE - PROGRESS UPDATE
Covering: July 7-8, 2026


JULY 7 - PAYROLL, ATTENDANCE & EMPLOYEE SELF-SERVICE
-----------------------------------------------------
- Payroll generation now stops at yesterday's date - it never pulls in
  today's timecards, since some employees haven't clocked out yet.
- Payroll periods now continue from the end of the last generated run
  instead of always using a fixed rolling window, and the end date rolls
  back to the nearest weekday if it lands on a Saturday or Sunday.
- Added a "Regenerate Last Payroll" option so an already-generated period
  can be recalculated with current data instead of being stuck.
- Payslip-ready notifications now show up in the bell icon right away.
  Root cause of the delay: Supabase Realtime had never actually been
  turned on for any table in the app, so "live" updates required a page
  reload everywhere, not just for payslips. Fixed with a migration.
- Time Correction request notifications are now scoped to just HR,
  Finance, and the employee who submitted the request - previously went
  to a much broader admin list.
- Time Correction requests can now also fix meal start/end times, not
  just clock-in/clock-out.
- Added a PTO eligibility rule: employees aren't eligible for PTO until
  they've completed 1 year with the company, based on their hire date.
- Truck Stock page is now paginated (100 rows at a time, same pattern as
  Ticket List) - it was lagging badly from loading everything at once.
- Added an In Stock / In Use status field to Truck Stock, and wired it
  into part lookup/fulfillment so a part marked "in use" is never
  suggested as available stock for a different ticket.
- Payroll now records time down to the second instead of rounding to the
  minute, so pay calculates accurately to the cent.
- Simplified payroll notifications to a generic "Payslip is Ready"
  message - no dollar amounts, no personal name as the sender.
- Rebuilt the payslip view: instead of one lump-sum "Regular Hours" row,
  it now shows a real day-by-day table (Date, Time In, Meal In, Meal Out,
  Time Out, Duty Hours, Rate, Total) with a grand total at the bottom.

- Rebuilt Part Inventory to show real parts ordered on real tickets
